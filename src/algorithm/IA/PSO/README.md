# 粒子群优化算法 (Particle Swarm Optimization)

## 算法概述

粒子群优化算法是一种模拟鸟群觅食行为的群体智能算法，通过粒子在搜索空间中的移动和信息共享来寻找全局最优解。

## 核心原理

1. **群体智能**：多个粒子协同搜索，共享信息
2. **个体认知**：粒子记住自己的历史最优位置
3. **社会认知**：粒子学习群体中最优位置
4. **速度更新**：根据惯性、个体经验和社会经验调整速度

## 算法流程

```
初始化粒子群 → 评估适应度 → 更新个体最优和全局最优 → 更新粒子速度和位置 → 重复直到满足停止条件
```

## 代码结构

- `Particle.java`: 粒子类，实现单个粒子的位置、速度和适应度管理
- `ParticleSwarmOptimization.java`: PSO主类，实现算法流程控制
- `FunctionOptimizationExample.java`: 函数优化应用示例

## 经典应用

### 1. 函数优化
- **单峰函数优化**: Sphere, Rosenbrock函数
- **多峰函数优化**: Rastrigin, Ackley函数
- **约束优化问题**: 处理各种约束条件的优化

### 2. 工程优化
- **结构优化**: 材料强度、重量优化
- **控制系统**: PID参数调优
- **电力系统**: 经济调度、负荷预测

### 3. 机器学习
- **神经网络训练**: 权重和阈值优化
- **特征选择**: 选择最优特征子集
- **超参数调优**: 模型参数自动优化

### 4. 组合优化
- **TSP问题变体**: 带时间窗的TSP
- **调度问题**: 作业车间调度
- **背包问题**: 多维度背包优化

### 5. 其他领域
- **图像处理**: 图像分割参数优化
- **通信网络**: 网络路由优化
- **金融投资**: 投资组合优化

## 参数说明

- **种群大小 (swarmSize)**: 粒子数量，通常20-50
- **惯性权重 (inertiaWeight)**: 控制粒子保持原有速度，通常0.4-0.9
- **认知权重 (cognitiveWeight)**: 个体经验学习因子，通常1.0-2.0
- **社会权重 (socialWeight)**: 群体经验学习因子，通常1.0-2.0
- **最大速度 (maxVelocity)**: 限制粒子移动速度，通常与搜索空间相关

## 速度更新公式

```
v_i(t+1) = w × v_i(t) + c1 × r1 × (pbest_i - x_i(t)) + c2 × r2 × (gbest - x_i(t))
x_i(t+1) = x_i(t) + v_i(t+1)
```

其中：
- w: 惯性权重
- c1: 个体认知权重
- c2: 社会认知权重
- r1, r2: 随机数 [0,1]
- pbest_i: 个体历史最优位置
- gbest: 全局最优位置

## 优势

- 算法简单，易于实现
- 参数少，易于调节
- 全局搜索能力强
- 收敛速度快
- 适用于连续优化问题

## 局限性

- 易陷入局部最优
- 对初始参数敏感
- 收敛精度有限
- 不适合离散优化问题

## 使用建议

1. **参数设置**:
   - 惯性权重: 从0.9线性递减到0.4
   - 认知和社会权重: 通常设为1.494
   - 种群大小: 20-40个粒子

2. **改进策略**:
   - 采用线性递减惯性权重
   - 使用约束处理机制
   - 结合局部搜索算法

3. **问题适配**:
   - 连续优化问题效果最好
   - 多峰问题需增大种群规模
   - 高维问题需适当增加迭代次数

5. **离散PSO**: 适用于组合优化
